# DB 설계 
 
 - RDBMS
 
 - Oracle 
   - DDL  - 데이터 정의어(Data Definition Language)
   - DCL - 데이터 제어어(Data Control Language)  사용자에게 권한 생성 혹은 권한 삭제 같은 명령어
   - DML  - 데이터 조작어(Data Manipulation Language)
   - TCL  - 트렌젝션 제어어(Transaction Control Language)
