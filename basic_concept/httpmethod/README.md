# HTTP METHOD 

### GET 

요청받은 URI의 정보를 검색하여 응답한다.
 
### POST

요청된 자원을 생성(CREATE)한다.

### PUT

요청된 자원을 수정(UPDATE)한다.

### PATCH

PUT과 유사하게 수정(UPDATE)할 때 사용한다. PATCH는 해당자원의 일부를 교체하는 의미로 사용.

### DELETE

요청된 자원을 삭제 한다.

### HEAD

메세지 헤더를 취득한다

### OPTIONS

제공하고 있는 메소드를 문의한다.
